### 知识点

  - 没有 return 语句的函数定义，Python会在末尾加上 return None；或者使用不带值的 return 语句，也会返回 None
  - Python 访问局部变量和全局变量的规则：当搜索一个变量时候，Python 先从局部作用域开始搜索，如果没有就按照作用域逐层寻找
  - 如果要在函数中修改全局变量，要使用 global 语句
